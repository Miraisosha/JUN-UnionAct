
【　ACCESS MDBの最適化　】
　・AccessMDBの最適化処理を行う為には、下記の参照設定が必要です。
　　１．参照設定
　　　　→ プロジェクト(P) → 参照の追加(R) → 「COM」 タブで
　　　　　「Microsoft Jet and Replication Objects 2.6 Library」 追加


【　C1FlexGrid　】
　・C1FlexGrid使用するには、下記の参照設定とツールボックスの追加が必要です。

　１．参照設定
　　　　→プロジェクト(P) → 参照の追加(R) → 「.NET」 タブで
　　　　　「ComponentOne C!FlexGrid Classic JPN」 追加
　　　　　「ComponentOne C!FlexGrid Designer JPN」 追加
　　　　　「ComponentOne C!FlexGrid JPN」（C:\Program Files\C1FlexGrid5\Bin\C1.Win.C1FlexGrid2.dll） 追加
　　　　　「ComponentOne C!FlexGrid JPN」（C:\Program Files\C1FlexGrid5\Bin\C1.Win.C1FlexGrid4.dll） 追加

　２．ツールボックスアイテム追加
　　　　→ ツール(T) → ツールボックス アイテム選択(X) → 「.NET Framesork コンポーネント」タブで
　　　　　C1FlexGridのチェックボックスにチェックを付けて「OK」ボタン

　３．ツールボックス表示
　　　　→ 表示(V) → ツールボックス(X)でツールボックス表示
　　　　　ツールボックス→ 「全般」メニューにC1FlexGridが表示され、使用できるようになります。
